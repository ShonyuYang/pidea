## 前言

你有沒有想過，一個巴掌大的玻璃罐裡，能裝下整個晝夜輪替？

[Raspberrarium](https://github.com/mommotti/raspberrarium) 是一個極簡但極美的開源專案：用 Raspberry Pi 搭配兩顆 WS2812 LED，在一個直徑 8 公分、高 13 公分的封閉玻璃罐裡，模擬出精確到「分鐘」的日出、正午、日落，以及跟隨真實天文資料的月相變化。不是固定排程的開關燈，而是每 10 秒重新計算一次的平滑色溫過渡。

這不只是一個 LED 專案，更是一座活生生的桌上生態系統。

## 專案概述

### 硬體清單

| 零件 | 說明 |
|------|------|
| Raspberry Pi 3B+（或任何有 GPIO 的 Pi） | 主控制器，作者原本就拿來當 NAS |
| WS2812 LED × 2 | 左右各一顆，模擬月相盈虧 |
| 跳線 3 條 | 接 3.3V/5V、GND、GPIO 12 |
| 小玻璃罐 | 8cm × 13cm，任何密封容器皆可 |
| 半透明塑膠片 | 作為光線擴散片，柔化 LED 點光源 |
| 透氣布料 | 蓋在瓶口下方，讓濕氣緩慢排出 |

生態缸本身的分層基底（由下而上）：碎石 → 樹皮 → 木炭 → 砂與灰 → 泥土 → 培養土。每一層都有功能——排水、過濾、保濕、固根。作者在[搭建指南網站](https://mommotti.github.io/raspberrarium-site)裡有完整的圖文教學。

### 軟體架構

整個專案就一支 Python 腳本 `raspberrarium.py`，依賴：

- **`astral`**：根據你的經緯度計算真實日出日落時間
- **`adafruit-circuitpython-neopixel`**：驅動 WS2812 LED
- **`python-dotenv`**：從 `.env` 讀取時區和座標設定

啟動方式極簡：

```bash
sudo python3 raspberrarium.py --brightness 3 --silent
```

設定 `.env` 填入你的時區和經緯度，就完成了。作者也提供了 systemd service 設定，開機自動啟動、崩潰自動重啟。

## 技術亮點

### 分鐘級的平滑光線過渡

這個專案最讓人驚豔的地方，不是「能模擬日夜」，而是**怎麼模擬**。

系統每 10 秒重新計算一次當前的光線狀態，根據真實天文資料產生連續的色溫變化。不是「6 點開燈、18 點關燈」那種粗暴切換，而是從黎明的微光到正午的暖白、再到黃昏的橘紅，每一刻都不同。如果你盯著看，根本察覺不到任何跳變。

### 雙 LED 月相表現法

只用兩顆 LED 就能表現月相，靠的是左右亮度差異：

- **上弦月** → 右邊 LED 較亮
- **下弦月** → 左邊 LED 較亮
- **滿月** → 兩顆全亮
- **新月** → 全暗

簡單到不行，但透過擴散片看起來效果出奇地好。作者還貼心提醒：如果你在南半球，把罐子轉 180 度就好。

### 三段亮度可調

`--brightness` 參數支援 1-3 級。作者特別註明低亮度（1-2 級）時，某些階段 LED 可能看起來像是關掉的——這不是 bug，是因為模擬的光線本來就很微弱（比如月牙階段）。

### 零維護封閉生態

瓶蓋上打了 8 個小孔：7 個用於通風和注水、1 個讓 LED 線材通過。蓋子下方鋪一層透氣布，讓濕氣能緩慢循環但不會快速蒸發。封好之後，偶爾用針筒從小孔注水就行，完全不需要打開罐子。

## 心得與延伸

這個專案的魅力在於它的**克制**。作者明確說了：這個 repo 只管光線模擬，不管植物選擇、不管環境控制。正因為範圍小，所以每個細節都打磨得很好。

幾個可以延伸的方向：

1. **換成 ESP32**：作者自己也提到原本想用 ESP32，如果你沒有閒置的 Pi，一塊 ESP32 加上 MicroPython 完全可以做到一樣的事，而且功耗更低
2. **加入溫濕度感測器**：放一顆 BME280 進去，搭配 OLED 小螢幕顯示缸內環境數據
3. **多缸同步**：如果你有好幾個生態缸，可以讓它們共享同一個光線週期，或者模擬不同時區（東京缸 vs 倫敦缸？）
4. **縮時攝影**：接一個 Pi Camera，每小時拍一張，記錄植物在模擬日夜下的生長變化

整體來說，這是一個非常適合週末動手的專案——硬體成本幾乎為零（如果你已經有 Pi），程式碼乾淨易讀，而且成品放在書桌上真的很療癒。
