## 前言

Polaroid 拍立得的魅力從來不在畫質。那種「按下快門、等待顯影、拿到一張獨一無二的實體照片」的儀式感，是手機相簿裡一萬張照片都換不來的。

問題是：一包 8 張的 Polaroid i-Type 底片要價約 NT$500（2025 年還漲了兩次價），每按一次快門就是 NT$60 飛出去。

立陶宛 Maker [atomicsandwich](https://atomicsandwich.com) 的解法很直接——把底片換成熱感應紙。一捲 50 公尺的感熱紙不到 NT$100，每張照片的耗材成本低於 NT$0.03。他用 Raspberry Pi Zero、一台小型熱感應印表機和 3D 列印外殼，做出了一台真正的「窮人的拍立得」。

不過作者自己也承認：零件加起來其實比最便宜的 Polaroid 相機還貴。但長期來看，每張照片幾乎免費——而且你擁有的是一台完全可以自己改造的相機。

## 專案概述

### 運作流程

整台相機的運作邏輯很單純：

1. **按下快門按鈕** → Pi Zero 透過 CSI 介面驅動相機模組拍照（2000×2000）
2. **影像處理** → Python 腳本根據亮度自動選擇最佳化演算法
3. **列印輸出** → 透過 USB 將處理後的影像送到熱感應印表機
4. **撕下照片** → 完成

從按下快門到拿到照片，大約幾秒鐘。

### 硬體清單

| 元件 | 說明 | 估算價格（NT$）|
|------|------|----------------|
| Raspberry Pi Zero W | 主控制器，含 Wi-Fi / BT | ~500 |
| Pi Camera Module | CSI 相機模組 | ~300 |
| 熱感應印表機（PT-310）| 58mm ESC/POS 印表機 | ~600–1,800 |
| 行動電源（拆機）| 提供 5V 電源 + 電池指示燈 | ~200 |
| 3D 列印外殼 | FreeCAD 設計，三明治結構 | 材料 ~50 |
| LED ×3 | 藍（電源）/ 綠（相機就緒）/ 紅（拍照中）| ~10 |
| 按鈕 ×2 + 開關 ×1 | 快門、關機、電源開關 | ~30 |
| **合計** | | **~1,700–2,900** |

### 三顆按鈕，三個功能

- 🔵 **快門鍵**（正面）：拍照 + 列印
- 🔄 **重印鍵**（背面）：重新列印上一張照片——「一張給我，一張給你」
- ⏻ **關機鍵**（側面）：安全關閉 Pi Zero（搭配實體電源開關斷電）

## 技術亮點

### 自適應亮度校正——五種演算法的排列組合

熱感應印表機只能印黑白，沒有灰階。這意味著影像的明暗對比處理至關重要——太亮的照片會變成一片白，太暗的會糊成一團黑。

作者用 Python + OpenCV 實作了一套根據亮度值自動切換的影像處理流程：

| 亮度範圍 | 處理方式 | 原理 |
|----------|----------|------|
| < 60（很暗）| 直方圖均衡化 | 把像素分佈拉開，暗部細節全部拉出來 |
| 60–90（偏暗）| Gamma 校正（γ=0.7）| 非線性提亮，保留中間調層次 |
| 90–110（正常偏暗）| CLAHE | 局部自適應均衡，避免過度增強 |
| 110–130（正常偏亮）| Gamma 校正 | 微調亮度分佈 |
| > 130（很亮）| 對比度拉伸 | 壓縮高光、提升暗部，找回對比 |

這五種演算法的選擇閾值是作者「trial and error」調出來的。看似粗暴，但對熱感應印表機這種只有黑白兩色的輸出裝置來說，反而比精緻的 AI 降噪更實用——你需要的不是「最好看」，而是「印出來看得清楚」。

### 拆行動電源當電源——附贈過熱保護

作者沒有自己組裝鋰電池 + DC-DC 轉換器 + 充電控制器，而是直接拆了一顆行動電源。理由很實在：買現成的行動電源比分開買零件便宜，而且裡面已經整合好所有電源管理電路。

更意外的收穫是：拆出來的行動電源 PCB 上有一顆 **NTC 熱敏電阻**，當電池溫度過高時會自動切斷電源。在一台塞滿了印表機（會發熱）和 Pi Zero 的緊湊外殼裡，這個免費的安全機制格外珍貴。

作者還特別提醒：拆行動電源時要戴護目鏡，因為一旦戳破鋰電池，「你需要的不是一桶水，而是一桶沙。」

### 3D 列印三明治外殼

外殼分成三層：
1. **前殼**：容納 Pi Zero、電池、LED 和快門按鈕
2. **中殼**：印表機本體
3. **後殼**：電源電路和重印按鈕

三層像三明治一樣疊合。作者用 FreeCAD 建模，列印後打磨、上底漆填補層紋、最後噴上黃黑配色。成品看起來像一台真正的消費性產品，而不是一堆裸線纏在一起的 prototype。

## 心得與延伸

### 為什麼這個專案值得關注

這個專案的程式碼只有一個 Python 檔案，大約 150 行。硬體就是 Pi Zero + 印表機 + 相機模組，沒有任何自製 PCB。但它的完成度非常高——從 3D 列印外殼的噴漆、LED 狀態指示、安全關機機制，到重印按鈕的巧思，每個細節都被認真對待。

更重要的是它示範了一個觀念：**「不完美」本身就是特色**。熱感應印表機的黑白點陣輸出、偶爾過曝或偏暗的照片、感熱紙隨時間褪色——這些「缺陷」反而賦予每張照片獨特的個性，就像 Polaroid 底片的色偏和暈影一樣。

### 每張照片的成本比較

| 方案 | 每張成本 |
|------|----------|
| Polaroid i-Type 底片 | ~NT$60 |
| Instax Mini 底片 | ~NT$20 |
| 熱感應紙（本專案）| < NT$0.03 |

長期使用的話，這台「窮人的拍立得」大約拍 50 張就能回本（相對於 Polaroid 底片的花費）。

### 可能的延伸方向

- **加入抖動演算法（Dithering）**：目前的影像處理是亮度校正後直接送印，如果加上 Floyd-Steinberg dithering，可以在黑白兩色中模擬出更多灰階層次，照片品質會大幅提升
- **Wi-Fi 遠端觸發**：Pi Zero 本身有 Wi-Fi，可以做一個簡單的 Web 介面，讓朋友用手機觸發拍照
- **貼紙版本**：Reddit 上有人用可以印貼紙的熱感應印表機做了類似專案，拍完直接撕下來貼——派對上的殺手級應用
- **搭配 LicheeRV Nano**：如果想要更小的體積和更低的功耗，可以考慮用 LicheeRV Nano 替代 Pi Zero，它的 CSI 相機介面同樣可以驅動攝像頭，而且體積只有 Pi Zero 的一半
- **加入濾鏡選擇**：用旋鈕切換不同的影像風格——高對比、復古、漫畫風等
