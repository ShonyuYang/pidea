## 前言

如果有一台印表機，印出來的東西注定會消失，你會用它印什麼？

Reddit 用戶 slartibartfist 給出了兩個答案：**靈感**和**嘲諷**。他用 ESP32、步進馬達和一顆廉價雷射二極體，打造了一台「短命印表機」（Ephemeral Printer）。它在夜光紙上用光寫字，關燈後文字會發出幽幽綠光——然後在大約一分鐘內逐漸消散。

這個專案在 r/arduino 拿下超過 2,300 個 upvotes 和 Mod's Choice 標籤，Hackster.io 也做了專題報導。它不只是一個好玩的 gadget，背後的機構設計和軟硬體整合都值得細看。

## 專案概述

### 運作原理

核心概念很簡單：**用雷射光照射夜光（蓄光）紙，被照到的區域會蓄積能量並發光**。但要把這件事做成「列印文字」，就需要一套精巧的機構。

slartibartfist 沒有走「用雷射直接描繪字形」的路線（那需要極精確的運動控制），而是借鑑了古老的**菊輪印表機（daisy wheel printer）**概念：

1. **字模輪盤**：3D 列印一個圓形遮罩，圓周上鏤空了完整的字母表
2. **選字馬達**：一顆步進馬達旋轉輪盤，將目標字母對準雷射光路
3. **雷射曝光**：雷射從輪盤後方射出，光線只能通過鏤空字母的形狀
4. **移動列印頭**：第二顆步進馬達驅動整個列印頭沿紙條橫向移動

每個字母的「列印」就是：轉到對的字母 → 開雷射 → 移動到下一格。整行印完後，夜光紙上就浮現出一行清晰的文字。

### 硬體清單

| 元件 | 說明 |
|------|------|
| ESP32 | 主控制器，負責邏輯與隨機文字生成 |
| 步進馬達 ×2 | 一顆轉字模輪盤，一顆移動列印頭 |
| 步進馬達驅動板 | 驅動步進馬達（作者提到需要冷卻時間） |
| 雷射二極體 | 低功率雷射指示器等級 |
| 3D 列印字模輪盤 | 菊輪式鏤空字母遮罩 |
| 夜光紙 | Glow-in-the-dark paper，蓄光材質 |
| 按鈕 ×2 | 「?」觸發靈感模式，「!」觸發嘲諷模式 |
| 3D 列印外殼 | 機構框架，主要用回收零件搭建 |

### 兩種模式

- **❓ Oblique Strategies 模式**：隨機顯示一句創意啟發語句，源自 Brian Eno 與 Peter Schmidt 的經典創意卡牌。當你陷入創作瓶頸時，按下問號按鈕，機器會給你一句打破僵局的提示。
- **❗ Insult 模式**：隨機生成一句嘲諷。作者說：「因為我有個小孩，這會讓他笑。」

## 技術亮點

### 菊輪機構的現代復活

菊輪印表機是 1970-80 年代的技術，以高品質字體著稱（比點陣印表機清晰得多）。slartibartfist 用 3D 列印重現了這個概念，但把墨帶換成了雷射光、把紙換成了夜光材料。這個轉換巧妙地解決了「低成本雷射無法精確描繪字形」的問題——你不需要控制光束的形狀，只需要控制遮罩的位置。

### 熱管理的偷懶哲學

> *「列印後鎖定 30 秒讓文字消退，同時也讓步進馬達驅動板降溫，這樣我就不用裝散熱片了。」*

這是典型的 maker 思維：與其加硬體解決散熱問題，不如把冷卻時間變成功能的一部分。30 秒的等待既是「讓文字消失」的必要時間，也是「讓硬體休息」的設計巧思。一個限制變成了兩個功能。

### 回收零件的 Upcycling 精神

Hackster.io 報導特別提到這台機器主要使用回收零件（salvaged components）打造。這不只是省錢，更是一種 maker 文化的體現：從舊設備中拆出馬達和電子元件，賦予它們新的生命。

## 心得與延伸

### 為什麼這個專案值得關注

這台印表機的魅力不在技術難度——ESP32 驅動兩顆步進馬達加一顆雷射，硬體門檻並不高。它的魅力在於**概念的完整性**：

- 「短命」不是 bug，是 feature
- 列印的內容（靈感 / 嘲諷）完美契合「轉瞬即逝」的形式
- 機構設計（菊輪）優雅地解決了技術限制
- 30 秒冷卻期同時服務了功能和工程需求

這是一個「所有限制都被轉化為設計元素」的教科書案例。

### 可能的延伸方向

- **Wi-Fi 連線**：ESP32 本身就有 Wi-Fi，可以接入 API 顯示天氣、新聞標題、或朋友傳來的訊息
- **更大的字模輪盤**：加入數字、標點、甚至中文部首（雖然這會讓輪盤變得很大）
- **多色夜光**：使用不同波長的雷射搭配不同顏色的夜光材料
- **互動裝置**：放在咖啡廳或展覽空間，讓訪客按按鈕獲取「今日運勢」
- **搭配 LicheeRV Nano**：用 RISC-V 開發板取代 ESP32，加入攝影鏡頭做表情辨識，根據你的心情選擇印出靈感還是嘲諷

### 估算成本

如果你想複製這個專案，主要成本大約在 **NT$500-800** 左右：
- ESP32 開發板：~NT$150
- 步進馬達 ×2 + 驅動板：~NT$200
- 雷射二極體模組：~NT$50
- 夜光紙：~NT$100
- 3D 列印耗材：~NT$50（如果你已有印表機）

當然，前提是你有一台 3D 印表機來製作字模輪盤和外殼。
