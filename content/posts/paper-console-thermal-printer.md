## 前言

在人人都在追求更大螢幕、更高解析度的時代，有人反其道而行——做了一台**完全沒有螢幕**的裝置。

Paper Console（PC-1）是一台手工打造的熱感印表機，外殼用胡桃木和黃銅製成，內建 Raspberry Pi Zero 2W。操作方式極度簡單：轉動黃銅旋鈕選頻道，按下按鈕，58mm 感熱紙就吐出你要的內容——天氣、新聞、數獨、迷宮、甚至一場文字冒險遊戲。

作者 Travis Miller 在轉行當開發者之前，做了六年的家具師傅。他說：「我希望硬體摸起來像一件真正的物品，而不是一個專案盒子。」

## 專案概述

### 硬體配置

| 元件 | 規格 |
|------|------|
| 核心 | Raspberry Pi Zero 2W |
| 印表機 | 58mm 熱感印表機 |
| 輸入 | 旋轉編碼器 + 按鈕（GPIO） |
| 外殼 | 手工胡桃木 + 黃銅面板 |
| 紙張 | 58mm × 30mm 感熱紙卷 |
| 電源 | 5V 5A |

### 軟體架構

- **後端**：Python + FastAPI，提供 RESTful API
- **前端設定 UI**：React，在區域網路內用手機或電腦開啟即可設定
- **無雲端、無訂閱**：所有資料在本地處理，API key 由使用者自行提供

### 16 種可列印模組

Paper Console 將模組分為三大類，每個頻道可自由組合多個模組：

**內容類（Content）**
- ☀️ 天氣預報（即時）
- 📰 新聞頭條 + QR Code 連結（即時）
- 📡 RSS 訂閱（即時）
- 📧 未讀信件摘要（即時）
- 🌙 天文資訊：日出日落、月相（本地）
- 📅 行事曆：日/週/月檢視（即時）
- 📜 歷史上的今天（本地）
- 💬 名言佳句，內建 5000+ 條（本地）
- ✍️ 日記提示（本地）

**遊戲類（Games）**
- 🧩 數獨，三種難度（本地）
- 🔲 迷宮（本地）
- 🗡️ 文字冒險遊戲，用旋鈕選擇劇情（本地）

**工具類（Utilities）**
- 📝 自訂文字/備忘錄（本地）
- 📱 QR Code 產生器（本地）
- 🔗 Webhook，串接任何 API（即時）
- 🖥️ 系統監控：IP、磁碟、記憶體、CPU（本地）

標記為「本地」的模組完全離線運作，不需要網路連線。

## 技術亮點

### 開箱即用的設定流程

Paper Console 的初次設定體驗設計得非常優雅：

1. 插上電源，印表機自動印出設定說明（包含 WiFi SSID 和密碼）
2. 用手機連上 `PC-1-Setup-XXXX` WiFi
3. 開啟 `http://10.42.0.1` 輸入家用 WiFi 資訊
4. 完成！之後用 `http://pc-1.local` 就能管理所有設定

整個過程不到兩分鐘，不需要裝任何 App，不需要建立雲端帳號。

### 8 頻道 × 模組堆疊

旋鈕有 8 個位置，每個位置可以載入多個模組。例如：
- 頻道 1：天氣 + 數獨 + 名言（早晨套餐）
- 頻道 2：新聞 + 信件摘要（工作模式）
- 頻道 3：迷宮（給小孩玩）

還支援**排程列印**——設定好時間，早上起床桌上就有一張今日天氣和頭條新聞。

### 長按互動設計

按鈕不只是「按一下印東西」：
- **長按 5 秒**：進入快捷選單（目錄、系統監控、重印設定、重置 WiFi 等）
- **長按 15 秒**：恢復原廠設定

### 程式碼品質

GitHub repo 已有 **142 顆星、735 次 commit**，程式碼結構清晰，有完整的測試腳本和開發環境設定。支援在一般電腦上模擬運行（印表機輸出顯示在終端機），方便開發和測試。

## 心得與延伸

Paper Console 讓我想到一個問題：**我們真的需要螢幕來接收所有資訊嗎？**

一張紙條上的天氣預報，比手機上的通知更不容易被忽略——它就在那裡，實體的、不會被滑掉的。早上撕下一張數獨配咖啡，比打開手機 App 少了太多干擾。

從技術角度來看，這個專案展示了幾個值得學習的設計決策：

- **離線優先**：16 個模組中有 9 個完全離線運作，這對 IoT 裝置是很重要的可靠性設計
- **無雲端依賴**：使用者自備 API key，沒有月費、不怕服務商倒閉
- **Web UI 設定**：不需要專屬 App，任何瀏覽器都能用，大幅降低使用門檻
- **mDNS**：用 `pc-1.local` 取代記 IP 位址，對非技術使用者很友善

如果你想自己做一台，軟體完全開源，只需要一塊 Pi Zero 2W 和一台 58mm 熱感印表機就能開始。外殼的部分⋯⋯就看你的木工功力了 😄

作者目前以 **$495 美元**限量販售 10 台手工成品，對於收藏級的桌上裝置來說，這個價格反映了手工胡桃木和黃銅的用料成本。

### 延伸想法

- 用 ESP32 取代 RPi 做更小的版本？（但模組豐富度會受限）
- 加入 e-ink 小螢幕顯示目前頻道名稱
- 結合 Home Assistant 做智慧家庭事件印表機
- 用 3D 列印外殼降低成本，做成 DIY Kit
